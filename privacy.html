Privacy Policy

Thoughtfully is designed to help users save birthdays, gift ideas, favorites, personal notes, and meaningful moments.

Information entered into the app is stored locally on the user's device unless otherwise stated.

Notifications are optional and are used for birthday reminders and gentle app reminders.

Photo access is optional and is used only if a user chooses to add a profile photo.

Location access is optional and is used only to show a weather-based greeting on the Home screen. The app still works if location access is denied.
